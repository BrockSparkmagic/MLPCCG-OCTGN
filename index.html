MLPCCG-OCTGN

My Little Pony Collectible Card Game Plugin for OCTGN

Feed URL: https://www.myget.org/F/mlpccg-octgn/ To install this game, go to OCTGN's Games Manager, click "Add Game Feed", then type "MLPCCG" into the "Name" field and input the above url into the "Feed URL" field.

My Little Pony Collectible Card Game by Enterplay.
